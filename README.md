# 💒 Свадебный Telegram-бот — Артём × Полина

Бот для управления гостями и RSVP на свадьбу.

## Как это работает

1. Вы добавляете гостей в бота → бот генерирует персональную ссылку
2. Гость открывает ссылку → видит сайт с **своим именем** и персональным текстом
3. На сайте кнопка «Подтвердить в Telegram» → открывает бота
4. Гость нажимает «Приду / Не смогу» → вы получаете уведомление

## Установка

### 1. Создайте бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Придумайте имя: `Свадьба Артёма и Полины`
4. Придумайте username: `artem_polina_wedding_bot` (должен заканчиваться на `bot`)
5. Скопируйте полученный **токен**

### 2. Настройте проект

```bash
# Склонируйте или скопируйте файлы
cd wedding-bot

# Установите зависимости
npm install

# Создайте .env файл
cp .env.example .env
```

### 3. Заполните .env

```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_telegram_id
SITE_URL=https://ваш-домен.com/wedding-invitation.html
```

> **Как узнать свой Telegram ID?**
> Запустите бота и отправьте `/myid`

### 4. Запустите

```bash
# Обычный запуск
npm start

# Режим разработки (автоперезапуск при изменениях)
npm run dev
```

## Команды бота

### Для админа

| Команда | Описание |
|---------|----------|
| `/addguest Имя \| Текст` | Добавить гостя с персональным текстом |
| `/addguest Имя` | Добавить гостя (без персонального текста) |
| `/guests` | Список всех гостей и их статусы |
| `/stats` | Статистика ответов |
| `/link` | Получить ссылку для гостя (выбор из списка) |
| `/link abc123` | Получить ссылку по ID |
| `/remove abc123` | Удалить гостя |
| `/broadcast Текст` | Разослать сообщение всем гостям в Telegram |
| `/myid` | Узнать свой Telegram ID |

### Примеры добавления гостей

```
/addguest Мария Иванова | Дорогая Маша, мы так рады, что ты будешь с нами!
/addguest Семья Петровых | Дорогие Петровы, ждём вас всей семьёй!
/addguest Алексей Смирнов | Лёха, будет весело — приходи!
/addguest Анна Козлова
```

## Размещение сайта

Файл `wedding-invitation.html` — это статический сайт. Его можно разместить:

- **GitHub Pages** (бесплатно)
- **Netlify** (бесплатно)
- **Vercel** (бесплатно)
- **Любой хостинг** с поддержкой статики

Не забудьте заменить `BOT_USERNAME` в HTML-файле на username вашего бота.

## Размещение бота

Бот — это Node.js-приложение. Его можно запустить на:

- **Railway** (бесплатный tier)
- **Render** (бесплатный tier)
- **VPS** (DigitalOcean, Hetzner и т.д.)
- **Свой компьютер** (для тестирования)
